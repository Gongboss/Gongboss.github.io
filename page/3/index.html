void ReadIr() interrupt 0
	{
		u8 j,k;
		u16 err;
		Time=0;					 
		delay(700);	//7ms
		if(IRIN==0)		//确认是否真的接收到正确的信号
		{	 
			err=1000;				//1000*10us=10ms,超过说明接收到错误的信号
			/*当两个条件都为真是循环，如果有一个条件为假的时候跳出循环，免得程序出错的时
			侯，程序死在这里*/	
			while((IRIN==0)&&(err>0))	//等待前面9ms的低电平过去  		
			{			
				delay(1);
				err--;
			} 
			if(IRIN==1)			//如果正确等到9ms低电平
			{
				err=500;
				while((IRIN==1)&&(err>0))		 //等待4.5ms的起始高电平过去
				{
					delay(1);
					err--;
				}
				for(k=0;k<4;k++)		//共有4组数据
				{				
					for(j=0;j<8;j++)	//接收一组数据
					{
						err=60;		
						while((IRIN==0)&&(err>0))//等待信号前面的560us低电平过去
						{
							delay(1);
							err--;
						}
						err=500;
						while((IRIN==1)&&(err>0))	 //计算高电平的时间长度。
						{
							delay(10);	 //0.1ms
							Time++;
							err--;
							if(Time>30)
							{
								return;
							}
						}
						IrValue[k]>>=1;	 //k表示第几组数据
						if(Time>=8)			//如果高电平出现大于565us，那么是1
						{
							IrValue[k]|=0x80;
						}
						Time=0;		//用完时间要重新赋值							
					}
				}
			}
			if(IrValue[2]!=~IrValue[3])
			{
				return;
			}
		}			
	}

接下来，就是进一步解析了~

¶起始码识别

首先延时7ms,接近低电平转高电平的界值。

delay(700);

然后开始逼近，使用的变量为err:

while((IRIN==0)&&(err>0))
{
	delay(1);
	err--;
}

从7ms到9ms的过程，信号都是低电平，因此有IRIN==0。若IRIN非0，则证明已经进入4.5ms的高电平阶段。同时，若信号非指定信号，10ms内便不会有高电平出现，err自减为0，则跳出循环。

另外，制作时必须的PS技能，鄙人也不能再无视了（不过，我还是喜欢用AE）。

¶毽飞刘琦赋

毽子飞自刘琦

茫茫草野，浩浩平原。牛羊闲步，飞鸟暂歇。挥长鞭而牧人心悦，放雄鹰而猎者悠闲。砾石微震，风呼渐急。日月不明，飞沙卷泰山之石；乾坤复倒，共工触不周之山。牛羊惊散，飞鸟无踪。牧者抬首，穿墨空似有微明；猎人回望，尽乌蒙如见弱光。忽闻鹤唳，一毽当空。燃天明黄沙尽褪，耀空亮双日复出。千里雨霁，万里云销。天狼何见，毽飞刘琦耳！

时维三月，序数八九。冰湖动而草木兴，百花笑而万鸟吟。携毽球于阡陌，访胜地于碧海。登石垂草，眺远望天。风抚四面，曦临八方。屈壮臂则跃跃欲试，催强腱而蠢蠢将动。

举柔羽，提彩旃，响金音，坠毽球。聚天地之灵气，旋施其上；汇日月之精华，俱融厥中。及至近地，轻巧巧其宛腿；待往中毽，速灵灵其欲飞。目光所望，毽之所向。卷沙飞石，席暗囊光。穿重云势不可挡，破浓雾速非能及。冲利剑鸡鸣山颠，发强弩鹤唳深渊。夜茫茫耀空飞陨，天沉沉炫目流光。折天柱泰山如毁，绝地维黄河将摧。乾坤易位，赤县震动。四渎倒流，万民稽首。毽飞牧野，附飞石轩辕车裂；羽至扶桑，顺劲风嫘祖丝断。越祝融榣山琴停，飞共工龙冢石震。日神羲和之车，摇摇欲坠；月仙望舒之船，危危将倾。潮汐齐天，沉雷轰地，火覆昆山，电掣极南。望西天五百罗汉，无壁可扶；看上清三大天尊，拂尘莫寻。

盛极必衰，泰极否来。渐行渐缓，徐飞徐坠。径三江三江欲躲，越五岳五岳如避。泰山石重，不及万分之一；举国聚鼎，莫比毽之一羽。虎狼凶兽，四散奔逃，雕鹰飞禽，早已无踪。轰然入地，百重沙遮天蔽日；积波推势，千卷浪啸海万丈。炫光渐暗，流火无存。山外九鼎，万民伏地。

伏更腊易，雪落枫生，三年尘静，五年方寒。彩旃犹在，片金尚存。万丈之渊，不测之谷。百里无骨，千里无生。磐石之上，一毽三羽。如凤始驻，似凰方歇。无非他物，毽飞刘琦耳！
