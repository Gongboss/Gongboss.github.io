“百里少侠”号小车设计之初步功能设计

¶简单测试

0x06,即0110，左转

0x09,即1001，右转

那么……

0x0a,即1010，为后退

0x05,即0101，为前进

😈

于是，继续解决红外问题：关键是switch分支与break的问题。

¶switch中的break

如果加break,在执行完此次case后，便会跳出switch。

而不加switch，从上往下case判断，一旦遇到合适的，便开始执行此次case，尔后非但不跳出，还把下面的所有case都执行一遍（好反人类）。之前小车老是不成功，就是中了无break的毒，每次按完按键，都停留在最后一个0x00上，这样怎会发动小车呢？

¶想要实现的功能

  1. 挡位，即通过delay步进来调整电机速度。
  2. 液晶显示，起码能显示个"百里少侠"。
  3. default显示无状态，让程序更完善。
  4. 转弯（稍后细说）。

¶转弯

目前一共有三种转弯方式：

  1. 左轮全速减速，右轮全速加速
  2. 右轮全速减速，左轮全速加速
  3. 左轮不动，右轮加速
  4. 右轮不动，左轮加速
  5. 左右轮均加速，且左轮转速小于右轮
  6. 左右轮均减速，且左轮转速大于右轮

¶下一步

  1. 通过程序及测试进一步写出前三种转弯方式moto二进制代码。
  2. 通过程序测验步进调速是否能成功，并尽量多分出挡位
  3. 一次就好，尽量减少烧录次数。

为了测试，不得不建立一个专供单片机的独立供电系统，用三节纽扣电池加一个二极管足矣。

¶初代码bug

一旦进入go(i)里的循环，中断无法使跳出循环。或者说，中断执行了。但由于go(i)运行未结束，便一直卡在这里。另外，独立的供电系统也不成功，考虑去掉二极管。

¶供电系统

现在不得不重新谋划单片机的供电系统了。

整个装置一共应有两套各自独立的供电系统，一套供给L298N用于电机驱动，另一套供给单片机用于控制。

现在的问题是，纽扣电池不能提供足够的电流，使单片机无法正常运行。而L298N占用了所有电池匣，后续研究可能面临无匣可用的境地。

解决方案就是用9v大电池再串两个5号电池给L298N供电，用杨老师摔坏的分贝仪电池槽装3节5号电池给单片机供电。当然供的是3.3V，所以要串一个二极管。

刚刚查到资料，9V电池电流小，几十毫安，无法驱动电机。开来，是时候祭出能装6节电池的大电池匣了！也是为了可持续发展嘛！

不对！此电池匣有3对是并联，输出电压只能是4.5V！

看来，得制作一个电池匣了。当然，下下策还是用掉那么多匣。

电池匣做不成，电焊技术又不到家，只好用下下策了😭

除夕随笔

发表于 2022-01-31 分类于 随笔 阅读次数： Valine：
本文字数： 443 阅读时长 ≈ 1 分钟

还是几年前的时候，除夕给我的印象就是红。

到傍晚的时候，院子里点亮了灯笼，红彤彤的一片。红色的窗花、红色的对联、红色的蜡烛、红色的供奉桌……

一家人围在一起包饺子，火炉里的火格外地旺。听着外面的炮声，看着春晚，那时真是身在福中不知福。如今生活方式急剧变化，年味都淡了。

晚饭后，家里人会在火炉里烤几个红薯。新出炉的红薯肉是鲜玉米色的，特别嫩。

我总是喜欢放炮，把大挂鞭拆成一个一个的点。一开始把炮放在花栏墙上用香小心翼翼地点，后来胆子渐渐大了起来，把炮拿在手上点着，然后甩出去。炮捻子点燃后会射出很好看的火星束，就像火树银花的颜色一样。

爷爷会磊旺火。除夕下午，在一个1m多高的油桶上，放一层砖、一层碳、一层玉米棒子、一层元宝。最上面用一小块碳压一张“旺气冲天”，再挂两条元宝。接神时点燃，火会窜得很高。

最喜欢初一的早上，天色朦胧，大门上的灯笼用喜庆的红渲染模糊的天色。大门前水泥路的尽头是青色的山，还有朦胧的桦树，以及天边微微的红色。眼前尽是雾，却是欢乐逝去的清明，很美。
