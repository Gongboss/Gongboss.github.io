发酵瓶研究日志

发表于 2022-03-27 分类于 天工开物 阅读次数： Valine：
本文字数： 1.8k 阅读时长 ≈ 2 分钟

制作一个发酵瓶。

目前手头有一个汽车车灯，上面标有“12V 60/55W”,考虑用这个来加热。所以，第一步就是降压。

¶降压

降压最初想到的是降压电容，但功率着实太低，所以暂不考虑。

可用的元件就只有变压器了。

手头一共三个变压器，但只有一个知道参数，输出的是18V300MA，不合要求，最终从另外两个中选择了一个变压器。

这个变压器是连在一个以前做过的电路上的，效果就是通电点亮一个LED，断电后仍能显示一段时间。用了桥式整流与三极管、电容，可能是个枕边方便灯吧。

先进行万用表测试，接在整流桥的输入端，读数7.56V上下波动。。应该是足够了。

然后是预实验，如图，感觉良好。可以启动。

¶单片机供电

接了一个充电器电路板，稍稍成功。

¶控制电路

控制电路的核心在继电器，这里选用4098型号的，是从初中物理实验箱里找到的。

这里选取3v、0.36W，需电流0.12A，即120mA。而单片机管脚输出电压30ma，需要用三极管放大。

三极管选用NPN型，2N3904。

上拉电阻用100欧，再去掉图中R2，进行面包板预实验。

实验发现3094不行，8050才行。

用万用表检测通路，一切良好。如图

焊入电路板

发现不能离开单片机板子，于是只好“拖家带口”了。

继电器再接上加热灯，算是成功了！

《阿甘正传》短评

发表于 2022-02-13 分类于 随笔 阅读次数： Valine：
本文字数： 345 阅读时长 ≈ 1 分钟

花了一个上午的时间读完这本书，说起来，比较失望。

这本书意欲塑造一位自强不息的愚人，但是好像只有在越战、橄榄球赛时，我方读出阿甘的自强不息精神。至于他的诸多成就，更像是命运的巧克力与与生俱来的“白痴学者”造就的。到后来见总统、上太空、丛林生活，作者明显是想讽刺，我却读出了更多的荒诞成分。也许，是我不了解美国人的黑色幽默，而只钟情于姜文的黑色幽默吧。

但是，做自己认为正确的事，这是我一定要学习的！

这本书的英文原版也很有风格。作者以白痴的角度写，不仅避免了大量生词，还有意造成大量拼写错误，可惜我不懂美国文化，无法理解这样做的美妙之处。

总之，常常沉迷于快节奏生活的我确实不太能静下心来体味《阿甘正传》的反讽艺术与阿甘的进取精神，或许等人生阅历更丰富时，我才会有更多像与《百年孤独》那样的共鸣吧！

¶简单测试

0x06,即0110，左转

0x09,即1001，右转

那么……

0x0a,即1010，为后退

0x05,即0101，为前进

😈

于是，继续解决红外问题：关键是switch分支与break的问题。

¶switch中的break

如果加break,在执行完此次case后，便会跳出switch。

而不加switch，从上往下case判断，一旦遇到合适的，便开始执行此次case，尔后非但不跳出，还把下面的所有case都执行一遍（好反人类）。之前小车老是不成功，就是中了无break的毒，每次按完按键，都停留在最后一个0x00上，这样怎会发动小车呢？

¶想要实现的功能

  1. 挡位，即通过delay步进来调整电机速度。
  2. 液晶显示，起码能显示个"百里少侠"。
  3. default显示无状态，让程序更完善。
  4. 转弯（稍后细说）。

¶转弯

目前一共有三种转弯方式：

  1. 左轮全速减速，右轮全速加速
  2. 右轮全速减速，左轮全速加速
  3. 左轮不动，右轮加速
  4. 右轮不动，左轮加速
  5. 左右轮均加速，且左轮转速小于右轮
  6. 左右轮均减速，且左轮转速大于右轮

¶初代码bug

一旦进入go(i)里的循环，中断无法使跳出循环。或者说，中断执行了。但由于go(i)运行未结束，便一直卡在这里。另外，独立的供电系统也不成功，考虑去掉二极管。

¶供电系统

现在不得不重新谋划单片机的供电系统了。

整个装置一共应有两套各自独立的供电系统，一套供给L298N用于电机驱动，另一套供给单片机用于控制。

现在的问题是，纽扣电池不能提供足够的电流，使单片机无法正常运行。而L298N占用了所有电池匣，后续研究可能面临无匣可用的境地。

解决方案就是用9v大电池再串两个5号电池给L298N供电，用杨老师摔坏的分贝仪电池槽装3节5号电池给单片机供电。当然供的是3.3V，所以要串一个二极管。

刚刚查到资料，9V电池电流小，几十毫安，无法驱动电机。开来，是时候祭出能装6节电池的大电池匣了！也是为了可持续发展嘛！

不对！此电池匣有3对是并联，输出电压只能是4.5V！

看来，得制作一个电池匣了。当然，下下策还是用掉那么多匣。
